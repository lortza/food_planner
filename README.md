# Food Planner

WIP food planning app.

* Ruby 2.5.0
* Rails 5.2
* Postgres
* RuboCop
* RSpec

## Features

* List of recipes & ingredients
* Display full recipes
* Display ingredient breakdown for meal prep

## Getting Started

* Fork & Clone
* Bundle
* rake db setup
* rake db seed
* run RuboCop: `rubocop`
* run tests: `bundle exec rspec`
